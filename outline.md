# 文法的定义

文法G（全称短语结构文法PSG）是一个四元式，

<center>G = ( ∑，V，S，P ) </center>

∑：终结符集合；

V：非终结符集合；

S：开始符，S ∈ V；

P：产生式集合。

---

# 产生式

产生式的形式：

<center>α → β</center>

α ∈ ( ∑ ∪ V )<sup>+</sup>，α 至少包含一个非终结符。

β ∈ ( ∑ ∪ V )<sup>*</sup>，β可以为 ε，当β为ε时，α→ε称为空串产生式。

一个产生式的左边可以是串。

第一个产生式的左边只能有一个符号：开始符号S。

A ⇒* αβ

A ⇒<sup>*</sup> αβ
